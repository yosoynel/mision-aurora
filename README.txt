Proyecto convertido para Visual Studio Code

Archivos principales:
- index.html
- style.css
- script.js

Cómo ejecutarlo:
1. Abre esta carpeta en Visual Studio Code.
2. Abre index.html.
3. Ejecuta con la extensión Live Server o haz doble clic en index.html para abrirlo en el navegador.
4. Presiona “Iniciar Animación”.

Notas:
- La animación dura aproximadamente 46 segundos. Las voces fueron ajustadas para hablar más despacio y sincronizar mejor con el texto de cada personaje.
- Usa Canvas API para dibujar y Web Speech API para las voces.
- En algunos navegadores las voces pueden tardar unos segundos en cargar o pedir interacción del usuario.
